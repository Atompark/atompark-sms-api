Atomic Bulk SMS Service API

This PHP API allows to send sms via Atomic Bulk SMS Sender

To activate API 3.0. usage, please login into Atomic SMS Sender account and in the Settings section -> Atomic SMS Sender tab choose “Yes” in the Active API 3.0.

Note! Unlike the XML interface in API 3.0 the test mode is activated by transferring the test=1 parameter. Please note that the current parameter is also used in the checksum generating, if exists.

To view examples of using the API, please visit the Bulk SMS Api page on the official website

To use API methods means to send an HTTP-request to the URL like: http://atompark.com/api/sms/3.0/METHOD?key=public_key&sum=CONTROL_SUM&arg1=ARG_1&argN=ARG_N
